## mars-roadmap.md

# Дорожная карта MARS

Фазы сформулированы как **план**; фактическое завершение фиксируется отдельно (без приписывания статуса «готово» здесь).

| Фаза | Содержание |
|------|------------|
| **1** | **Web-GPT source pack** — документация и термины в репозитории/рабочей копии |
| **2** | **Cursor local starter** — локальный стартовый пакет для разработки (отдельный артефакт) |
| **3** | **Git repository** — формализация репо, ветвление, защита веток |
| **4** | **Agent registry** — живой реестр в коде/доках, процесс обновления |
| **5** | **Gulp Frontend Agent migration** — карточка, промпты, примеры, связка с CI |
| **6** | **Memory / logging** — lifecycle, audit, связка с хранилищем |
| **7** | **Storage adapters** — Postgres/Sheets/Files/S3-hybrid по политике |
| **8** | **n8n / OpenRouter** — интеграции и роутинг моделей по политике |
| **9** | **Реальный пилотный проект** — сквозной сценарий с evals и HITL |

Текущий пакет `web-gpt-sources` относится к **фазе 1**.
